# Side projects

Each application under this folder is **independent**: its own frontend and backend (or CLI), with separate dependencies and run instructions.

| Project | Description |
|--------|-------------|
| `issue-detector` | Requirement analyzer service (FastAPI) and MCP scaffolding — see its README. |
| `intelligent-calendar-agent` | Stateful calendar assistant with Google Calendar (or mock), long-term memory, evaluation suite, and web UI. |

There is no shared monorepo tooling between projects unless you add it.
